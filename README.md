# 🌐 Saloni Rawat - Personal Portfolio

This is my personal portfolio website built using **React.js and Tailwind CSS** to showcase my projects, technical skills, certifications, and achievements as a Computer Science student.

The website highlights my work in **Software Development, Data Science, and Machine Learning**, and provides an easy way for recruiters and collaborators to connect with me.

---

## 🚀 Features

- Modern and responsive UI
- Interactive sections for skills, projects, certifications, and achievements
- Smooth navigation with clean layout
- Contact form integration using Formspree
- Resume preview and download
- Professional training and education timeline
- GitHub and LinkedIn integration

---

## 🛠️ Tech Stack

- **React.js**
- **Tailwind CSS**
- **Vite**
- **JavaScript**
- **Formspree (Contact Form)**

- Home
